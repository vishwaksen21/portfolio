## 🚀 Getting Started

### Prerequisites
- Node.js 20+ and npm

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/vishwaksen21/portfolio.git
   cd portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run development server**
   ```bash
   npm run dev
   ```

4. **Open in browser**
   ```
   http://localhost:3000
   ```

### Build for Production

```bash
npm run build
npm start
```
